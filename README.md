# 🧬 DNA Analyzer

A Python-based DNA sequence analyzer using Biopython.

## Features
- Nucleotide base count (A, T, G, C)
- Sequence length validation
- GC% and AT% calculation
- Complementary strand generation
- Reverse complement
- DNA to RNA transcription
- RNA to Protein translation

## Technologies Used
- Python
- Biopython

## How to Run
1. Install Biopython: `pip install biopython`
2. Run: `python dna_analyzer.py`
3. Enter DNA sequence when prompted

## Example Input
ATGAAACCCGGGTTTTAA
Any valid DNA sequence using A, T, G, C bases
Minimum length: 30 bases recommended
